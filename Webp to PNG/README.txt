L'utilizzo del presente programma prevede due modalità di conversione:

1. Conversione singola: è possibile convertire un singolo file attraverso l'interfaccia del programma, trascinando l'immagine da convertire nella finestra del software e premendo il tasto "invio".

2. Conversione multipla: è possibile convertire più file contemporaneamente mediante l'inserimento di tutte le immagini in formato .webp presenti nella loro cartella di origine. L'avvio del programma avvierà automaticamente il processo di conversione, ed i file convertiti saranno salvati nella cartella di destinazione specificata dal programma stesso, denominata "Png".
